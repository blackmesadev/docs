    Command Quick Reference

    Moderation

    All commands in this module require the moderation permission node. Subsequent commands will require the moderation.<command> permission node.

    ban <target:user[]> [time:duration] [reason:string...]

    Ban target for time if specified from the server for reason.

    If time is not specified, the ban will be permanent.

    Example

    • !ban 96269247411400704 Did something stupid
    • !ban 96269247411400704 206309860038410240 5d Did something stupid

    kick <target:user[]> [reason:string...]

    Kick target from the server for reason.

    Example

    • !kick 96269247411400704 Did something stupid

    mute <target:user[]> [time:duration] [reason:string...]

    Mute target for time if specified for reason.

    If time is not specified, the mute will be permanent.

    Example

    • !mute 96269247411400704 Did something stupid
    • !mute 96269247411400704 206309860038410240 5d Did something stupid

    search [target:user]

    Search the infractions from a target.

    If target is not specified, the command author is used (see guild.searchself).

    Example

    • !search 96269247411400704

    softban <target:user[]> [reason:string...]

    Softban (ban then unban) target from the server for reason.

    If delete is not specified, it will default to 1 day.

    Example

    • !softban 96269247411400704 3d Did something stupid

    unban <target:user[]> [reason:string...]

    Unban target from the server for reason.

    Example

    • !unban 96269247411400704 206309860038410240 All is forgiven

    unmute <target:user[]> [reason:string...]

    Unmute target from the server for reason.

    Example

    • !unmute 96269247411400704 206309860038410240 All is forgiven

    Administration

    All commands in this module require the admin permission node. Subsequent commands will require the admin.<command> permission node.

    purge [type:string] [filter:string...] [messages:number]

    Purge messages messages from the current channel that match type and filter.

    Filter types

    • bots - Only messages from bots
    • string - Only messages that contain the specified string at filter
    • users - Only messages from users
    • all - All messages

    Assumptions / Shortcuts

    • If type is not specified, all is assumed.

    • If type is specified as an ID, the bot will assume its type based on what the ID is for. (For example, if the ID is for a user, the bot will assume users and the ID will be added to filter.)

    • If messages looks like a user ID, the bot will assume it is a user and add it to filter.

    • If messages is not specified, 100 is assumed.

    Example

    • !purge 20 - Purge 20 messages from the current channel.
    • !purge bots 20 - Purge 20 messages from the current channel that are from bots.
    • !purge string naughty 20 - Purge 50 messages from the current channel that contain the word naughty.

    deepsearch [target:user]

    Deep Search the infractions from a target. This shows all infractions, including those that have expired. If target is not specified, the command author is used.

    Note: guild.searchself will grant if target is the command author, the user has implied access to their expired infractions.

    Example

    • !deepsearch 96269247411400704

    duration [uuid:uuid] [duration:duration]

    Set the duration of an infraction from the point the update is made. If uuid is not specified, the duration will be applied to the most recent infraction made by the command author.

    Note: moderation.updateself will grant if uuid is not specified or the specified infraction was made by the command author.

    Example

    • !duration 20m
    • !duration c488c9ec-423a-437c-817c-41d0d4141e58 1d

    expire [uuid:uuid] [duration:duration]

    Expire an infraction early. If uuid is not specified, the duration will be applied to the most recent infraction made by the command author.

    Note: moderation.updateself will grant if uuid is not specified or the specified infraction was made by the command author.

    Example

    • !expire c488c9ec-423a-437c-817c-41d0d4141e58

    reason [uuid:uuid] [reason:string...]

    Set the reason of an infraction. If uuid is not specified, the reason will be applied to the most recent infraction made by the command author. Note: moderation.updateself will grant if uuid is not specified or the specified infraction was made by the command author.

    remove <uuid:uuid>

    Remove an infraction from the database. Unlike the update commands, uuid is required in remove. Note: moderation.removeself will grant if the specified infraction was made by the command author.

    Example

    • !remove c488c9ec-423a-437c-817c-41d0d4141e58

    Guild

    All commands in this module require the guild permission node. Subsequent commands will require the guild.<command> permission node.

    The commands in this module are typically safe to expose to your users.

    guildinfo

    Get information about the current server.

    Example

    • !guildinfo

    userinfo [target:user]

    Get the user information of a given target. If target is not specified, the command author is used. Note: guild.userinfoself will grant if target is the command author.

    Example

    • !userinfo 96269247411400704

    botinfo

    Get information about the bot.

    Example

    • !botinfo
